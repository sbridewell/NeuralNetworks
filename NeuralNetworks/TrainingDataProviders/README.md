# TrainingDataProviders folder

A training data provider represents the question that a neural network is trying to answer. It provides inputs and expected outputs which can be used to train a neural network and to test a trained network. The data could be generated programmatically, or it could be read from a file, database or any other data store that might be available.

This folder also contains helper classes which may be specific to a particular training data provider, for example a representation of a quadratic equation which can help to generate training data for a neural network that is trying to learn how to solve quadratic equations.
